## Technical Specifications

### Video Format
- **Container**: MP4 (H.264 encoding preferred)
- **Resolution**: 1080p (1920x1080) minimum, 720p acceptable for standups
- **Frame Rate**: 30 fps recommended, 24 fps minimum
- **Audio**: AAC codec, 128 kbps minimum bitrate

### Audio Requirements
- **Microphone**: Headset microphone strongly recommended
- **Environment**: Quiet recording space, minimal background noise
- **Volume**: Consistent audio levels throughout recording
- **Clarity**: Clear articulation, appropriate speaking pace

### File Management
- **Naming Convention**: Follow exact patterns specified above
- **File Size**: Optimize for GitHub storage (< 100MB per file preferred)
- **Backup**: Keep local copies before uploading to repository
- **Version Control**: Use `_v1`, `_v2` suffixes for iterations

## Recording Tools (Recommended)
